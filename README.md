# 副業支援エージェント - プロジェクト全体ガイド

**プロジェクト ID**: 018  
**プロジェクト名**: Side Hustle Support Agent  
**目的**: デジイナ大学メンバーが副業で月5万円以上の収益化を実現できるようサポート

### 3. 🔄 ワークフロー設計
**ファイル**: [`WORKFLOW_DESIGN.md`](./WORKFLOW_DESIGN.md)

**各ワークフローには以下が含まれる**:
- フローチャート（視覚的表現）

- エラーハンドリング

**対象読者**: UX/UIデザイナー、プロダクトマネージャー

**読むべき順序**: 3番目（ユーザー体験設計）

## 📊 プロジェクト概要

### 対象ユーザー

| セグメント | 特徴 | ペルソナ例 |
|-----------|------|----------|
| **初心者** | 副業未経験 | 会社員、「月5万円稼ぎたい」 |
| **中級者** | 副業経験1-3ヶ月 | フリーランス初心者、「単価を上げたい」 |
| **上級者** | 副業経験3ヶ月以上 | 既に稼いでいる、「新しいスキルを開拓」 |

### 5つのコアワークフロー

```
1️⃣ 初回カウンセリング
   └─ スキルから最適な副業を発見（5～10分）

2️⃣ 段階別サポート
   └─ 初期→成長→安定の3段階で実行計画を提供

3️⃣ ポートフォリオ作成
   └─ Upwork・Coconala対応プロフィール文を生成

4️⃣ 営業戦略プラン
   └─ 営業ルート別に具体的なアクション計画を提示

5️⃣ 月次フォローアップ
   └─ 進捗を分析し、改善提案を提供
```

### 利用可能なツール

| ツール | 用途 | 利用頻度 |
|--------|------|--------|
| **Web検索** | 市場情報・相場調査 | 初回：高、継続：中 |
| **WebFetch** | プラットフォーム分析 | 初回：高 |
| **ローカルファイル** | 既存知識の参照 | 毎回 |
| **Claude API** | テキスト生成 | 毎回（提案文・記事など） |

### 成功指標

```
エージェント全体:
  ✅ ユーザー満足度: 80%以上
  ✅ 提案実行率: 50%以上
  ✅ 月収達成率: 70%以上（3ヶ月後）

各ユーザー:
  初心者: ポートフォリオ完成 (1週間), 初案件 (2週間)
  中級者: 単価向上 1.5倍以上 (3ヶ月後)
  上級者: 月定額契約 3社以上
```

## 💡 実装時の重要な考慮点

### 1. スケーラビリティ
```
現状: 月50ユーザー対応
目標: 月200ユーザー対応（3ヶ月後）

対応:
  - Web検索並列実行（3同時実行まで）
  - Claude API キューイング
  - キャッシュの有効利用
```

### 2. エラーハンドリング
```
優先順位:
  1️⃣ Web検索失敗 → キャッシュから返す
  2️⃣ API制限 → キューイング
  3️⃣ ローカルファイル不在 → スキップして続行
  4️⃣ 不正入力 → エラーメッセージ + 再入力
```

### 3. セキュリティ
```
実装時に確認:
  ✅ ユーザー情報の暗号化
  ✅ API キーの環境変数化
  ✅ ログの個人情報マスキング
  ✅ アクセス制御（認証・認可）
```

### 4. パフォーマンス
```
目標応答時間:
  初回カウンセリング: 5～10分以内
  テンプレート生成: 2～3分以内
  月次フォローアップ: 3～5分以内
  
最適化ポイント:
  - Web検索の並列実行
  - キャッシュの活用
  - API呼び出しの最小化
```
